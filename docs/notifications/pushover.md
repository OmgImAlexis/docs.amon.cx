<h2 id="pushover">Pushover</h2>

<h3>In Pushover:</h3>
<ol>
	<li>On your Pushover Dashboard, click <strong>Register an Application</strong>, select a Name and click <strong>Create Application</strong>.
		<img src="/assets/img/screenshots/pushover-create-app.png" alt="">
	</li>
	<li>Copy the application<strong>API Token</strong> and the <strong>User Key</strong> from your dashboard.
		<img src="/assets/img/screenshots/pushover-app-key.png" alt="">
		<img src="/assets/img/screenshots/pushover-user-key.png" alt="">
	</li>
</ol>

<h3>In Amon:</h3>
<ol>
	<li>In your Amon instance, in Settings, select the <strong>"Notifications"</strong> tab, 
		paste the application API Token and User Key from Pushover and press <strong>'Save'</strong>
		<img src="/assets/img/screenshots/amon-pushover.png" alt="">
	</li>

	<li>
		That is it. The next time you Add or Edit your alerts, you can select the Pushover checkbox and 
		the next time Amon triggers an alert, it will send the data to Pushover
		<img src="/assets/img/screenshots/amon-add-notification.png" alt="">
	</li>
</ol>
